> AI-powered search engine that combines real-time web retrieval with LLM reasoning

```
User Query → Tavily Search (N results) → Gemini 2.5 Flash → Cited Answer
```

- 🗄️ **Redis caching** for fast repeated queries
- 📜 **Search history** endpoint — last 5 searches persisted
- 🐳 **Dockerized** and deployed on Hugging Face Spaces
